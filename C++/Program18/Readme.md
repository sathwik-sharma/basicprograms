## Problem

```bash
Write a function which prints first num1 terms of the series 3n+2 which are not multiples of num2. 
```

## Take Input
```bash
Number 1(num1)
Number 2(num2) 
0 < num1 < 100 & 0 < num2 < 100  
```

## Sample Input

```bash 
num1 = 10;
num2 = 4;
```
## Sample Output

```bash
5
11
14
17
23
26
29
35
38
41
```
